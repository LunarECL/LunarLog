# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| < Latest | ❌       |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report security issues by emailing the maintainer directly or by using [GitHub's private vulnerability reporting](https://github.com/LunarECL/LunarLog/security/advisories/new).

We will acknowledge receipt within 48 hours and provide a timeline for a fix.
